# Demo Script

## 3-5 Minute Portfolio Demo

1. Open `http://localhost:3000`.
2. Log in as `analyst@example.com` with `LocalAnalyst123!`.
3. Upload files from `demo-data`.
4. Process each document.
5. Ask: "Summarize the main AI infrastructure risks across the uploaded annual reports."
6. Show the answer, citation cards, confidence badge, retrieved evidence, and trace panel.
7. Ask a weak or unsupported question to trigger human review.
8. Log in as `reviewer@example.com`.
9. Approve or edit the review item.
10. Log in as `admin@example.com`.
11. Run evaluations.
12. Show analytics and audit logs.
13. Explain local-only architecture.
